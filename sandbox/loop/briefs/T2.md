# 任务简报: T2 CLI 入口
- 尝试次数: 1

## 全局上下文
用 Node.js 编写一个命令行温度换算小工具，纯标准库，无任何 npm 依赖。
- 源码放在 src/ 下
- 代码注释使用中文

## 任务目标
基于 T1 的换算模块实现 CLI：接受一个形如 `100C` 或 `32F` 的参数，输出换算结果。

## 开工前必读
无

## 上游交接
[来自 T1] 关键决策: 换算函数为纯数学换算, 不做输入校验/取整; 非数字输入返回 NaN, 参数校验由 CLI 层负责; 返回原始浮点数不四舍五入, 格式化交给下游。
[来自 T1] 对下游的提醒:
- 接口约定：`const { c2f, f2c } = require('./src/convert.js')`
  - `c2f(c: number): number` 摄氏 → 华氏
  - `f2c(f: number): number` 华氏 → 摄氏
- CommonJS 模块，不能用 import 语法直接引入
- 结果未做精度处理，CLI 展示时如需固定小数位请自行 toFixed

## 允许写入的路径
- src/cli.js

## 验收标准
- `node src/cli.js 100C` 输出包含 212
- `node src/cli.js 32F` 输出包含 0
- 参数缺失或格式非法时输出中文用法提示, 退出码为 1

## 约束
无
